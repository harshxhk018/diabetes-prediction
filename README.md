# Diabetes Prediction

Machine Learning project to predict diabetes using patient health parameters.

## Dataset
- diabetes.csv

## Algorithms Used
- Logistic Regression
- Random Forest

## Results
- Logistic Regression Accuracy: 70.8%
- Random Forest Accuracy: 77.9%

## Files
- diabetes.csv
- diabetes_prediction_18.ipynb
- diabetes_model.pkl
